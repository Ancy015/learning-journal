# Learning Journal

This repository contains daily Java practice files (`day01.java` to `day08.java`).

## How to run

Use a Java JDK (17+ recommended). Example for `day06.java`:

```powershell
# From the repo root
javac day06.java
java day06
```

## Notes

- Java files are kept simple and standalone.
- Build artifacts and IDE files are ignored via `.gitignore`.
